# 项目文件描述


- api/sdk/src/main/java/com/ke/bella/openapi/protocol/completion/CompletionResponse.java: 这个文件定义了CompletionResponse类，继承自OpenapiResponse。包含Choice和TokenUsage嵌套类，用于描述响应结果的详细信息和token使用情况。与CompletionLogHandler类交互，提供响应的数据结构。

- api/sdk/src/main/java/com/ke/bella/openapi/protocol/completion/StreamCompletionResponse.java: 这个文件定义了StreamCompletionResponse类，继承自OpenapiResponse，表示一个流式的完成响应。包含的关键类有内部静态类Choice，用于表示选择项。该文件与消息工具的调用紧密相关，涉及到ToolCall等类。

- api/sdk/src/main/java/com/ke/bella/openapi/protocol/completion/Message.java: 这个文件定义了Message类，表示完成消息的结构，支持多角色和工具调用，包含函数调用和内容的定义。

- api/server/src/main/java/com/ke/bella/openapi/protocol/completion/CompletionLogHandler.java: 这个文件实现了CompletionLogHandler类，用于处理端点日志。主要功能是计算请求和响应的token使用量，以及处理完成响应的时间度量。与CompletionResponse类交互，处理端点为/v1/chat/completions。

- api/server/src/main/java/com/ke/bella/openapi/protocol/completion/ResponseHelper.java: 这个文件定义了ResponseHelper类，提供了完成响应的数据处理和转换功能。

- api/server/src/main/java/com/ke/bella/openapi/protocol/completion/OpenAIAdaptor.java: 用于适配OpenAI协议完成请求的适配器类，实现了流式及非流式任务请求。

- api/server/src/main/java/com/ke/bella/openapi/protocol/completion/callback/StreamCompletionCallback.java: 这个文件实现了StreamCompletionCallback类，负责处理流式完成响应的回调。涉及到的关键功能包括安全检查和日志记录，与EndpointProcessData和ISafetyCheckService等类有重要交互。

- api/server/src/main/java/com/ke/bella/openapi/endpoints/ChatController.java: ChatController类提供聊天相关的REST API端点，处理完成请求，使用AdaptorManager等类进行协议适配。

- api/sdk/src/main/java/com/ke/bella/openapi/protocol/completion/CompletionSseListener.java: 实现了CompletionSseListener类，用于处理SSE事件流中的完成消息，通过Callback接口进行回调。

- api/server/src/main/java/com/ke/bella/openapi/protocol/completion/callback/StreamCallbackProvider.java: StreamCallbackProvider类提供流式回调的生成，支持消息和完成回调，与SseEmitter等类交互。

- api/sdk/src/main/java/com/ke/bella/openapi/utils/JacksonUtils.java: JacksonUtils提供JSON序列化和反序列化的工具方法，包含日期处理和格式自定义功能。

45     private Map<String, Object> countMetrics(long startTime, long startMills, long endTime, long firstPackageTime, CompletionResponse.TokenUsage usage) {
46         int inputToken = usage.getPrompt_tokens();
47         int outputToken = usage.getCompletion_tokens();
48         int ttft = 0;
49         if(firstPackageTime != 0) {

56     private CompletionResponse.TokenUsage countTokenUsage(CompletionRequest request, OpenapiResponse openapiResponse, String encodingType) {
57         if(openapiResponse.getError() != null) {
58             int httpCode = openapiResponse.getError().getHttpCode();
59             if(httpCode > 399 && httpCode < 500 && httpCode != 408) {
60                 CompletionResponse.TokenUsage tokenUsage = new CompletionResponse.TokenUsage();
61                 tokenUsage.setPrompt_tokens(0);
62                 tokenUsage.setCompletion_tokens(0);
63                 tokenUsage.setTotal_tokens(0);
64                 return tokenUsage;
65             }
66         }
67         CompletionResponse response = null;
68         if(openapiResponse instanceof CompletionResponse) {
69             response = (CompletionResponse) openapiResponse;
70         }
71         if(response != null && response.getUsage() != null) {
72             return response.getUsage().validate();
73         }
74         EncodingType encoding = EncodingType.fromName(encodingType).orElse(EncodingType.CL100K_BASE);
75         //计费模型请求消耗量
76         //计算非userMessage的token用量
77         int requestToken = 0;
78         List<String> textMessage = new LinkedList<>();
79         List<Pair<String, Boolean>> imgMessage = new LinkedList<>();
80         if(request.getMessages() != null) {
81             for (Message message : request.getMessages()) {
82                 if(CollectionUtils.isNotEmpty(message.getTool_calls())) {
83                     textMessage.addAll(getToolCallStr(message.getTool_calls()));
84                 } else {
85                     //如果message.getContent()是String类型
86                     if(message.getContent() instanceof String) {
87                         textMessage.add((String) message.getContent());
88                     } else if(message.getContent() instanceof java.util.List) {
89                         for (Map content : (java.util.List<Map>) message.getContent()) {
90                             if(content.containsKey("text")) {
91                                 textMessage.add((String) content.get("text"));
92                             } else if(content.containsKey("image_url")) {
93                                 //如果包含类型为string的image_url
94                                 if(content.get("image_url") instanceof String) {
95                                     imgMessage.add(Pair.of((String) content.get("image_url"), false));
96                                 } else if(content.get("image_url") instanceof Map) {
97                                     String url = (String) ((Map) content.get("image_url")).get("url");
98                                     boolean lowResolution = "low".equals(((Map) content.get("image_url")).get("detail"));
99                                     imgMessage.add(Pair.of(url, lowResolution));
100                                 }
101                             }
102                         }
103                     }
104                 }
105             }
106         }
107         Optional<Integer> userTextMessageToken = textMessage.stream().map(x -> TokenCounter.tokenCount(x, encoding)).reduce(Integer::sum);
108         Optional<Integer> userImgMessageToken = imgMessage.stream().map(x-> TokenCounter.imageToken(x.getLeft(), x.getRight())).reduce(Integer::sum);
109         requestToken += userTextMessageToken.orElse(0) + userImgMessageToken.orElse(0);
110

114                         return getToolCallStr(x.getMessage().getTool_calls());
115                     } else {
116                         return Lists.newArrayList(x.getMessage().getContent());
117                     }
118                 }).flatMap(List::stream)
119                 .map(String.class::cast)
120                 .map(x -> TokenCounter.tokenCount(x, encoding)).reduce(Integer::sum).orElse(0);
121         CompletionResponse.TokenUsage tokenUsage = new CompletionResponse.TokenUsage();
