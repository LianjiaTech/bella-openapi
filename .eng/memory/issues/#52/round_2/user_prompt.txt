# 项目文件描述


- api/README.md: 这个文件是Bella OpenAPI的README，主要介绍了API对多模型请求的支持功能，用户可以在请求中指定多个模型。如果首选模型不可用，系统将尝试其他模型。文件中包含如何使用多模型的指导。

- api/server/src/main/java/com/ke/bella/openapi/endpoints/ChatController.java: ChatController类定义了用于处理聊天完成请求的REST接口，包含请求路由、安全检查、协议适配及响应逻辑。通过Spring框架实现REST控制器功能。

- api/server/src/main/java/com/ke/bella/openapi/protocol/ChannelRouter.java: 负责渠道路由选择的逻辑，包括筛选和选择可用的渠道，主要类是ChannelRouter，与ChannelService和ModelService交互。

- api/server/src/main/java/com/ke/bella/openapi/protocol/AdaptorManager.java: 定义了AdaptorManager类，用于管理不同协议适配器的注册和获取操作。通过endpoint和protocol识别适配器，并提供相应的服务支持。

- api/sdk/src/main/java/com/ke/bella/openapi/protocol/completion/CompletionRequest.java: 这个文件定义了CompletionRequest类，主要用于构建开放API的请求。它实现了UserRequest接口，并包含多个字段如model、user、messages等，用于配置API请求的参数和行为。

- api/server/src/main/java/com/ke/bella/openapi/protocol/completion/MultiModelSupport.java: 这个Java文件定义了MultiModelSupport类，主要用于验证请求中指定的模型数量是否合法。类中包含validateModelCount和getMaxModelsPerRequest方法，负责模型数量验证及获取每次请求的最大模型数量。

- api/server/src/main/java/com/ke/bella/openapi/protocol/completion/CompletionAdaptor.java: 定义CompletionAdaptor接口，用于处理补全请求，支持普通和流式补全。关联CompletionProperty类，用于异步或同步数据补全。

- api/server/src/main/java/com/ke/bella/openapi/endpoints/EmbeddingController.java: 这个文件实现了EmbeddingController控制器，负责处理/v1/embeddings端点请求。它与路由、适配器和限流管理器交互。

- api/server/src/main/java/com/ke/bella/openapi/protocol/embedding/EmbeddingAdaptor.java: 定义Embedding协议适配器接口，提供embedding请求的处理方法，指定端点为/v1/embeddings。

- api/server/src/main/java/com/ke/bella/openapi/endpoints/AudioController.java: AudioController.java实现了与音频处理相关的REST接口，包括实时语音识别、文本转语音和音频转录功能。主要使用的类和接口有ChannelRouter、AdaptorManager、LimiterManager等，通过这些接口来管理音频流和协议适配器，并处理与其他服务的交互。

58     public Object completion(@RequestBody CompletionRequest request) {
59         String endpoint = EndpointContext.getRequest().getRequestURI();
60         String model = request.getModel();
61         EndpointContext.setEndpointData(endpoint, model, request);
62         boolean isMock = EndpointContext.getProcessData().isMock();
63         ChannelDB channel = router.route(endpoint, model, EndpointContext.getApikey(), isMock); // This now handles multiple models
64         EndpointContext.setEndpointData(channel);
65         if(!EndpointContext.getProcessData().isPrivate()) {
66             limiterManager.incrementConcurrentCount(EndpointContext.getProcessData().getAkCode(), model);
67         }
68         Object requestRiskData = safetyCheckService.safetyCheck(SafetyCheckRequest.Chat.convertFrom(request,
69                     EndpointContext.getProcessData(), EndpointContext.getApikey()), isMock);
70         EndpointContext.getProcessData().setRequestRiskData(requestRiskData);
71         EndpointProcessData processData = EndpointContext.getProcessData();
72         String protocol = processData.getProtocol();
73         String url = processData.getForwardUrl();
74         String channelInfo = channel.getChannelInfo();
75         CompletionAdaptor adaptor = adaptorManager.getProtocolAdaptor(endpoint, protocol, CompletionAdaptor.class);
76         CompletionProperty property = (CompletionProperty) JacksonUtils.deserialize(channelInfo, adaptor.getPropertyClass());
77         if(isMock) {
78             fillMockProperty(property);
79         }
80         adaptor = decorateAdaptor(adaptor, property, processData);
81 
82         EndpointContext.setEncodingType(property.getEncodingType());
83         if(request.isStream()) {
84             SseEmitter sse = SseHelper.createSse(1000L * 60 * 5, EndpointContext.getProcessData().getRequestId());
85             adaptor.streamCompletion(request, url, property, StreamCallbackProvider.provide(sse, processData, EndpointContext.getApikey(), logger, safetyCheckService, property));
86             return sse;
87         }
88 
89         CompletionResponse response = adaptor.completion(request, url, property);
90         Object responseRiskData = safetyCheckService.safetyCheck(SafetyCheckRequest.Chat.convertFrom(response, EndpointContext.getProcessData(), EndpointContext.getApikey()), isMock);
91         response.setSensitives(responseRiskData);
92         response.setRequestRiskData(requestRiskData);
93         return response;
94     }

106     private CompletionAdaptor<?> decorateAdaptor(CompletionAdaptor<?> adaptor, CompletionProperty property, EndpointProcessData processData) {
107         if(StringUtils.isNotBlank(property.getQueueName())) {
108             if(adaptor instanceof CompletionAdaptorDelegator) {
109                 adaptor = new QueueAdaptor<>((CompletionAdaptorDelegator<?>)adaptor, jobQueueService, jobQueueService, processData);
110             } else {

112             }
113         }
114         if(property.isFunctionCallSimulate()) {

117         return adaptor;
118     }

48     public ChannelDB route(String endpoint, String model, ApikeyInfo apikeyInfo, boolean isMock) {
49         if(StringUtils.isBlank(endpoint) && StringUtils.isBlank(model)) {
50             throw new BizParamCheckException("没有可用渠道");
51         }
52         
53         if (model != null && model.contains(",")) {
54             // Process multiple models
55             return routeMultipleModels(endpoint, model, apikeyInfo, isMock);
56         } else {
57             // Single model routing
58             return routeSingleModel(endpoint, model, apikeyInfo, isMock);

63      * Routes request with multiple models
64      */
65     private ChannelDB routeMultipleModels(String endpoint, String modelString, ApikeyInfo apikeyInfo, boolean isMock) {
66         String[] models = modelString.split(",");
67         
68         // Check if number of models exceeds the maximum allowed
69         if (models.length > maxModelsPerRequest) {
70             throw new BizParamCheckException("请求模型数量超过最大限制: " + maxModelsPerRequest);
71         }
72         
73         // For multiple models, we'll select the first available model's channel
74         for (String model : models) {
75             try {
76                 ChannelDB channel = routeSingleModel(endpoint, model.trim(), apikeyInfo, isMock);
77                 if (channel != null) {
78                     return channel;
79                 }
80             } catch (Exception e) {
81                 // Continue to next model if this one fails
82                 continue;
83             }
84         }
85         
86         throw new BizParamCheckException("没有可用渠道，所有指定的模型都无法访问");
87     }

92     private ChannelDB routeSingleModel(String endpoint, String model, ApikeyInfo apikeyInfo, boolean isMock) {
93         List<ChannelDB> channels;
94         String entityCode;
95         if (model != null) {
96             String terminal = modelService.fetchTerminalModelName(model);
97             entityCode = terminal;
98             channels = channelService.listActives(EntityConstants.MODEL, terminal);
99         } else {
100             entityCode = endpoint;
101             channels = channelService.listActives(EntityConstants.ENDPOINT, endpoint);
102         }
103         if (CollectionUtils.isEmpty(channels)) {
104             if(isMock) {
105                 return mockChannel(null);
106             } else {
107                 throw new BizParamCheckException("没有可用渠道");
108             }
109         }
110         if(!isMock) {
111             channels = filter(channels, entityCode, apikeyInfo);
112         }
113         channels = pickMaxPriority(channels);
114         ChannelDB channel = random(channels);
115         return isMock ? mockChannel(channel) : channel;
116     }

118     /**
119      * 1、筛选账户支持的数据流向（风控） 2、筛选可用的渠道
120      *
121      * @param channels
122      *
123      * @return
124      */
125     private List<ChannelDB> filter(List<ChannelDB> channels, String entityCode, ApikeyInfo apikeyInfo) {
126         Byte safetyLevel = apikeyInfo.getSafetyLevel();
127         String accountType = apikeyInfo.getOwnerType();
128         String accountCode = apikeyInfo.getOwnerCode();
129         List<ChannelDB> filtered = channels.stream()
130                 .filter(channel -> !EntityConstants.PRIVATE.equals(channel.getVisibility()) ||
131                         (accountType.equals(channel.getOwnerType()) && accountCode.equals(channel.getOwnerCode())))
132                 .filter(channel -> getSafetyLevelLimit(channel.getDataDestination()) <= safetyLevel)
133                 .collect(Collectors.toList());
134         if(CollectionUtils.isEmpty(filtered)) {
135             if(LOWEST_SAFETY_LEVEL.equals(safetyLevel)) {
136                 filtered = channels.stream().filter(this::isTestUsed)
137                         .collect(Collectors.toList());
138             }
139             if(CollectionUtils.isEmpty(filtered)) {
140                 throw new ChannelException.AuthorizationException("未经安全合规审核，没有使用权限");
141             }
142             if(freeAkOverload(EndpointContext.getProcessData().getAkCode(), entityCode)) {
143                 throw new ChannelException.RateLimitException("当前使用试用额度,每分钟最多请求" + freeRpm + "次, 且并行请求数不能高于" + freeConcurrent);
144             }
145         }
146         Set<String> unavailableSet = metricsManager.getAllUnavailableChannels(
147                 filtered.stream().map(ChannelDB::getChannelCode).collect(Collectors.toList()));
148         filtered = filtered.stream()
149                 .filter(channel -> channel.getDataDestination().equals(EntityConstants.PROTECTED) ||
150                         channel.getDataDestination().equals(EntityConstants.INNER) ||
151                         !unavailableSet.contains(channel.getChannelCode()))
152                 .collect(Collectors.toList());
153         if(CollectionUtils.isEmpty(filtered)) {
154             throw new ChannelException.RateLimitException("渠道当前负载过高，请稍后重试");
155         }
156         return filtered;
157     }

159     private boolean isTestUsed(ChannelDB channel) {
160         return 1 == channel.getTrialEnabled() && !EntityConstants.PRIVATE.equals(channel.getVisibility());
161     }
162 
163     private boolean freeAkOverload(String akCode, String entityCode) {
164         return limiterManager.getRequestCountPerMinute(akCode, entityCode) >= freeRpm
165                 || limiterManager.getCurrentConcurrentCount(akCode, entityCode) >= freeConcurrent;
166     }

168     private Byte getSafetyLevelLimit(String dataDestination) {
169         switch (dataDestination) {
170         case EntityConstants.PROTECTED:
171             return 10;
172         case EntityConstants.INNER:
173             return 20;
174         case EntityConstants.MAINLAND:
175             return 30;
176         case EntityConstants.OVERSEAS:
177             return 40;
178         }
179         return 40;
180     }
181 
182     private List<ChannelDB> pickMaxPriority(List<ChannelDB> channels) {
183         List<ChannelDB> highest = new ArrayList<>();
184         String curVisibility = EntityConstants.PUBLIC;
185         String curPriority = EntityConstants.LOW;
186         for (ChannelDB channel : channels) {
187             String priority = channel.getPriority();
188             String visibility = StringUtils.isNotBlank(channel.getVisibility()) ? channel.getVisibility() : EntityConstants.PUBLIC;
189             int compare = compare(priority, curPriority, visibility, curVisibility);
190             if(compare < 0) {
191                 continue;
192             }
193             if(compare > 0) {
194                 highest.clear();
195                 curPriority = priority;
196                 curVisibility = visibility;
197             }
198             highest.add(channel);
199         }
200         return highest;
201     }
202 
203     private int compare(String priority, String curPriority, String visibility, String curVisibility) {
204         if(visibility.equals(curVisibility)) {
205             if(priority.equals(curPriority)) {
206                 return 0;
207             }
208             if(priority.equals(EntityConstants.LOW)) {
209                 return -1;
210             }
211             if(priority.equals(EntityConstants.NORMAL)) {
212                 if(curPriority.equals(EntityConstants.HIGH)) {
213                     return -1;
214                 }
215             }
216             return 1;
217         } else {
218             return visibility.equals(EntityConstants.PRIVATE) ? 1 : -1;
219         }
220     }

222     private ChannelDB random(List<ChannelDB> list) {
223         if(list.size() == 1) {
224             return list.get(0);
225         }
226         int rand = random.nextInt(list.size());
227         return list.get(rand);
228     }

+# Bella OpenAPI

+## Multi-Model Support
+
+This API supports specifying multiple models in a single request. This allows the system to try different models if the first choice is unavailable.
+
+### How to Use
+
+When making a completion request, you can specify multiple models in the `model` field using comma-separated values:
+
+```json
+{
+  "model": "gpt-4,gpt-3.5-turbo,claude-2",
+  "messages": [
+    {"role": "system", "content": "You are a helpful assistant."},
+    {"role": "user", "content": "Hello!"}
+  ]
+}
+```
+
+The system will try to use the models in the order specified. If the first model is unavailable, it will try the second, and so on.
+
+### Configuration
+
+The maximum number of models that can be specified in a single request is configurable via the application.yml file:
+
+```yaml
+bella:
+  openapi:
+    max-models-per-request: 3
+```
+
+The default maximum is 3 models per request.
